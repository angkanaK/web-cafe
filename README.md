# ☕ Booklynz Cafe - Web Ordering System

โปรเจกต์ฝึกเขียนเว็บไซต์ร้านคาเฟ่ที่เน้นการฝึกใช้ **Semantic HTML** และ **Modern CSS (Flexbox & Grid)** เพื่อสร้างโครงสร้างหน้าเว็บที่ถูกต้องตามมาตรฐานและรองรับการใช้งานจริง

## 🚀 ฟีเจอร์ที่ทำเสร็จแล้ว (Current Progress)
- [x] **Semantic HTML5:** แบ่งสัดส่วน Header, Nav, Main, Section, Aside อย่างถูกต้อง
- [x] **Responsive Header:** ระบบนำทางที่ใช้ Flexbox จัดระเบียบ Logo และ Menu
- [x] **Product Grid Layout:** โครงสร้างสำหรับแสดงรายการอาหารและตะกร้าสินค้า
- [x] **Git Workflow:** การจัดการเวอร์ชันของโค้ดผ่าน GitHub

## 🛠️ เทคโนโลยีที่ใช้ (Tech Stack)
- **HTML5:** โครงสร้างเว็บแบบ Semantic
- **CSS3:** การจัดเลย์เอาต์ด้วย Flexbox
- **Git & GitHub:** สำหรับ Version Control
