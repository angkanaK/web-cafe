# ☕ Booklynz Cafe - Web Ordering System

โปรเจกต์ฝึกเขียนเว็บไซต์ร้านคาเฟ่ที่เน้นการฝึกใช้ **Semantic HTML** และ **Modern CSS (Flexbox & Grid)** เพื่อสร้างโครงสร้างหน้าเว็บที่ถูกต้องตามมาตรฐานและรองรับการใช้งานจริง

## 🚀 ฟีเจอร์ที่ทำเสร็จแล้ว (Current Progress)
- [x] **Semantic HTML5:** แบ่งสัดส่วน Header, Nav, Main, Section, Aside อย่างถูกต้อง
- [x] **Responsive Header:** ระบบนำทางที่ใช้ Flexbox จัดระเบียบ Logo และ Menu
- [x] **Product Grid Layout:** โครงสร้างสำหรับแสดงรายการอาหารและตะกร้าสินค้า
- [x] **Git Workflow:** การจัดการเวอร์ชันของโค้ดผ่าน GitHub

## 🌟 อัปเดตล่าสุด (Recent Updates)
- [x] **Dynamic Image Styling:** จัดการรูปภาพอาหารให้สวยงามและสม่ำเสมอด้วย `object-fit: cover`
- [x] **Interactive Shopping Cart:** พัฒนาระบบตะกร้าสินค้าด้วย JavaScript (Vanilla JS)
- [x] **DOM Manipulation:** ระบบเพิ่มรายการอาหารลงในตะกร้าแบบ Real-time
- [x] **Calculation Logic:** ระบบคำนวณยอดเงินรวม (Total Price) อัตโนมัติเมื่อมีการสั่งซื้อ

## 🧠 ทักษะที่ได้ฝึกฝน (Skills Gained)
- **JavaScript DOM API:** การใช้ `querySelectorAll`, `createElement`, และ `appendChild`
- **Event Handling:** การจัดการ Event `click` เพื่อตอบสนองต่อผู้ใช้งาน
- **Data Parsing:** การจัดการข้อมูล String และแปลงเป็น Number ด้วย `parseInt()` และ `.replace()`
- **Problem Solving:** การแก้ปัญหา Layout พังจากการโหลดรูปภาพและการลำดับไฟล์ JavaScript
